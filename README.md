ASP.NET Core MVC Practice Project
This project was created for learning and practicing ASP.NET Core MVC, Entity Framework Core, and general web development concepts.
It is not intended to be a production-ready application, but rather a space to experiment, build features, and improve my skills.

Purpose of the Project
The main goals of this project:

Learn the structure of ASP.NET Core MVC
Practice controllers, views, models, and routing
Work with Entity Framework Core and databases
Practice dependency injection and services
Understand how to organize a clean folder structure
Experiment with UI/UX using Bootstrap
Practice authentication/authorization (Identity)
Technologies Used
ASP.NET Core MVC
Entity Framework Core
PostgreSQL
Bootstrap
C#
.NET 9
